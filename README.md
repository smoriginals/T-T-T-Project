# About 

## Noughts and Crosses (Tic Tac Toe)

A simple and interactive Noughts and Crosses (Tic Tac Toe) game built using HTML, CSS, and JavaScript.
This project demonstrates core front-end development concepts such as DOM manipulation, event handling, responsive layout, and basic game logic—all without using any external libraries or frameworks.
This project recreates the classic Noughts and Crosses game for the web. It is designed to be lightweight, easy to understand, and visually clean. The game supports two players playing on the same device and provides instant feedback for wins and draws.

The project was created for learning, practice, and demonstration purposes, making it suitable for beginners exploring JavaScript-based game logic.


# Features

- Two-player gameplay (X vs O)
- Win and draw detection
- Clean and responsive UI
- Smooth user interaction
- Reset and new game functionality
- No external dependencies
- Built With
- HTML5 – Structure
- CSS3 – Styling and layout
- JavaScript (Vanilla) – Game logic and interactivity
